## 电脑怎么"做判断"

生活里我们每天都在做判断："如果下雨，就带伞；否则不带。""如果分数够 60，就及格；不够就补考。"

这种"**如果……就……否则……**"的思路，编程里叫**条件判断（conditional）**。程序会先看一个条件成不成立，再决定接下来走哪条路。这一章的主角，就是实现它的关键词 `if`（英语"如果"）。

不过在学 `if` 之前，我们得先认识一种全新的"值"，它是所有判断的基础。

## 真还是假：布尔值

前面我们见过数字（`int`、`float`）和字符串（`str`）。现在来认识第四种、也是最简单的一种值——**布尔值（boolean）**。

布尔值只有两个：**`True`（真）** 和 **`False`（假）**。你可以把它理解成一个只能回答"是"或"否"的开关：要么开（True），要么关（False），没有中间状态。

> 小贴士：`True` 和 `False` 首字母**必须大写**，而且不加引号。写成 `true` 或 `"True"` 都是错的。它俩是 Python 里的特殊词。

布尔值是从哪来的？最常见的来源，是**比较两个东西**。我们来看看怎么比较。

## 比较运算符

想问电脑"3 比 5 大吗？"，就用**比较运算符（comparison operator）**。Python 一共有 6 个：

```python
# compare.py
print(3 > 5)     # 3 大于 5 吗？
print(3 < 5)     # 3 小于 5 吗？
print(5 == 5)    # 5 等于 5 吗？
print(5 != 3)    # 5 不等于 3 吗？
print(5 >= 5)    # 5 大于或等于 5 吗？
print(3 <= 2)    # 3 小于或等于 2 吗？
```

运行结果：

```
False
True
True
True
True
False
```

看，每个比较都得出一个布尔值：成立就是 `True`，不成立就是 `False`。把这 6 个符号读熟：

| 符号 | 意思 | 例子 | 结果 |
|------|------|------|------|
| `>`  | 大于 | `7 > 3` | `True` |
| `<`  | 小于 | `7 < 3` | `False` |
| `==` | 等于 | `4 == 4` | `True` |
| `!=` | 不等于 | `4 != 5` | `True` |
| `>=` | 大于或等于 | `6 >= 6` | `True` |
| `<=` | 小于或等于 | `6 <= 5` | `False` |

字符串也能比较，最常用的是判断"相不相等"：

```python
# compare_str.py
password = "python"
print(password == "python")   # 输入的密码对吗？
print(password == "java")
```

运行结果：

```
True
False
```

## 天大的坑：`==` 和 `=` 不是一回事

请画三条横线记住这一点，它是本章最容易栽跟头的地方：

> **一个等号 `=` 是"赋值"（把右边装进左边），两个等号 `==` 是"比较"（问左右相不相等）。**

它们长得像，作用天差地别：

- `age = 13` 的意思是：**把 13 装进 `age` 这个盒子**。这是命令。
- `age == 13` 的意思是：**问一句，`age` 里装的是不是 13？** 这是提问，会得到 `True` 或 `False`。

一个是"我给你 13"，一个是"你是 13 吗？"。想判断相不相等，一定要用**两个**等号 `==`。写成一个等号，程序要么报错，要么行为完全不对。这个错新手几乎都会犯，你现在提前知道了。

## if：如果……就……

准备工作做完，主角登场。`if` 让程序**在条件成立时，才执行某段代码**：

```python
# if_basic.py
score = 85

if score >= 60:
    print("恭喜，及格了！")
```

运行结果：

```
恭喜，及格了！
```

我们来拆开这段代码，它有几个必须记牢的部分：

- **`if`** 后面跟一个**条件**（这里是 `score >= 60`），也就是一个能算出 `True` 或 `False` 的式子。
- 条件后面跟一个**英文冒号 `:`**，这个冒号绝不能少，它在说"下面开始是条件成立要做的事"。
- 冒号下面那些**往右缩进**的行，就是"条件成立时才执行"的代码。

如果把 `score` 改成 50 再运行呢？`50 >= 60` 是 `False`，条件不成立，缩进里的 `print` 就**被跳过**，程序什么也不打印。你可以自己改改试试。

## 缩进：Python 靠它分辨"哪些代码属于 if"

这里要专门讲一个 Python 特别看重的东西——**缩进（indentation）**，也就是每行代码前面空的那几格。

很多语言用大括号 `{ }` 来圈出"属于 if 的代码"，Python 不用，它靠**缩进**。缩进进去的，就属于上面那个 `if`；顶回到最左边的，就不属于了。看这个对比：

```python
# indent_demo.py
score = 50

if score >= 60:
    print("这行缩进了，属于 if，只在及格时打印")
print("这行没缩进，不属于 if，永远都会打印")
```

运行结果：

```
这行没缩进，不属于 if，永远都会打印
```

看，`score` 是 50，不及格，所以缩进的第一句被跳过了；但第二句没缩进，它不归 `if` 管，照样打印。

> 小贴士：缩进用 **4 个空格**是 Python 的通行习惯。大多数编辑器里，你按一下 **Tab 键**就会自动缩进。同一段代码的缩进要对齐，不能一会儿 2 格一会儿 4 格，否则会报错。

## else：否则……

只有"如果"还不够。我们常常想说"如果……**否则**……"——条件成立走一条路，不成立走另一条路。这就要用到 **`else`**（英语"否则"）：

```python
# if_else.py
score = 50

if score >= 60:
    print("恭喜，及格了！")
else:
    print("差一点，下次加油！")
```

运行结果：

```
差一点，下次加油！
```

`else` 也要跟一个冒号，下面同样要缩进。它的意思是："上面那个 `if` 的条件**不**成立时，就执行我这里。" `if` 和 `else` 就像一个岔路口：条件成立走左边，不成立走右边，**两条路只会走一条**。

## elif：多个岔路口

`if/else` 只能分两条路。可现实里常常有好几种情况，比如成绩要分"优、良、中、差"四档。这时用 **`elif`**（是 "else if" 的缩写，意思是"否则，如果……"）：

```python
# grade.py
score = 85

if score >= 90:
    print("评级：优")
elif score >= 80:
    print("评级：良")
elif score >= 60:
    print("评级：中")
else:
    print("评级：差")
```

运行结果：

```
评级：良
```

程序是**从上往下**一个个检查条件的：

1. 先问 `score >= 90`？85 不够 90，`False`，跳过。
2. 再问 `score >= 80`？85 够，`True`！打印"良"，然后**整个判断结束**，后面的 `elif`、`else` 都不再看了。

这就是关键规律：**一串 `if / elif / else` 里，只会执行其中一个分支**——从上往下遇到的第一个成立的条件。所以顺序很重要：因为 85 同时满足 `>= 80` 和 `>= 60`，但它先撞上 `>= 80`，就归到"良"了。

> 小贴士：`elif` 可以写任意多个，夹在 `if` 和 `else` 中间。而 `if` 一串里，开头的 `if` 只有一个，结尾的 `else` 最多一个（也可以没有）。

## 组合条件：and、or、not

有时一个条件不够用。比如"周末**并且**天气好，才去公园"，这里有两个条件要同时满足。Python 用三个词来组合条件：**`and`（并且）**、**`or`（或者）**、**`not`（不是）**。

**`and`**：两个条件**都**成立，结果才 `True`（像"既要……又要……"）：

```python
# logic_and.py
age = 15
has_ticket = True

if age >= 12 and has_ticket:
    print("可以进场看电影")
else:
    print("不能进场")
```

运行结果：

```
可以进场看电影
```

年龄够 12（True），而且有票（True），两个都成立，`and` 的结果才是 True。只要有一个不成立，就进 `else`。

**`or`**：两个条件**只要有一个**成立，结果就 `True`（像"要么……要么……"）：

```python
# logic_or.py
day = "周六"

if day == "周六" or day == "周日":
    print("今天是周末，放假！")
else:
    print("今天要上学。")
```

运行结果：

```
今天是周末，放假！
```

**`not`**：把结果**反过来**，True 变 False，False 变 True（像"不是"）：

```python
# logic_not.py
is_raining = False

if not is_raining:
    print("没下雨，出去玩！")
```

运行结果：

```
没下雨，出去玩！
```

`is_raining` 是 `False`（没下雨），`not is_raining` 就把它翻成 `True`，于是打印。

## 完整代码一：成绩评级机

把 `input`、`int`、`if/elif/else` 全用上，做一个真正能用的评级程序。新建 `grade_machine.py`：

```python
# grade_machine.py

print("=== 成绩评级机 ===")

score = int(input("请输入你的分数："))   # 记得转成整数才能比较大小

if score >= 90:
    grade = "优"
    comment = "太棒了，学霸就是你！"
elif score >= 80:
    grade = "良"
    comment = "不错，继续加油！"
elif score >= 60:
    grade = "中"
    comment = "及格啦，还有提升空间。"
else:
    grade = "差"
    comment = "别灰心，下次一定行！"

print(f"{score} 分，评级：{grade}。{comment}")
```

运行结果（下划线部分是你输入的）：

```
=== 成绩评级机 ===
请输入你的分数：85
85 分，评级：良。不错，继续加油！
```

看，同一个程序，你输入不同的分数，它会走不同的分支，说出不同的话。程序真的会"看情况办事"了。注意这里每个分支都把结果存进 `grade` 和 `comment` 两个变量，最后统一用一句 `print` 打印——这样代码更整齐。

## 完整代码二：电影年龄检查器

再做一个用到组合条件的程序。有些电影有年龄限制，我们来做个检查器。新建 `movie_check.py`：

print("=== 电影年龄检查器 ===")

    print("抱歉，你还不够年龄，且没有家长陪同，不能观看。")

## 常见错误

**1. 把 `==` 写成了 `=`**

```python-norun
score = 60
if score = 60:
    print("正好及格线")
```

报错：

```
SyntaxError: invalid syntax
```

判断相不相等要用**两个**等号 `==`。一个等号是赋值，不能放在 `if` 的条件里。这是本章头号大坑，报错时先检查这里。

**2. 忘了写冒号 `:`**

```python-norun
score = 85
if score >= 60
    print("及格")
```

报错：

```
SyntaxError: expected ':'
```

`if`、`elif`、`else` 后面都必须跟一个英文冒号 `:`。报错信息 `expected ':'`（"缺一个冒号"）已经告诉你答案了。

**3. 缩进错了**

```python-norun
score = 85
if score >= 60:
print("及格")
```

报错：

```
IndentationError: expected an indented block after 'if' statement
```

`IndentationError`（缩进错误）表示：`if` 冒号下面那行忘了缩进。属于 `if` 的代码必须往右缩进（通常 4 个空格）。

**4. `and` / `or` 写成了中文或符号**

```python-norun
age = 15
if age > 12 && age < 18:
    print("青少年")
```

报错：

```
SyntaxError: invalid syntax
```

Python 里组合条件用的是英文单词 `and`、`or`、`not`，不是别的语言里的 `&&`、`||`，也不是中文的"并且""或者"。记牢这三个词。

## 本章小结

这一章，你的程序装上了"脑子"，会看情况做选择了。回顾要点：

- **布尔值**只有 `True`（真）和 `False`（假）两个，首字母大写、不加引号。
- **比较运算符** `>`、`<`、`==`、`!=`、`>=`、`<=` 会得出一个布尔值。
- **最大的坑：`=` 是赋值，`==` 是判断相等**，判断时千万别少写一个等号。
- **`if`** 在条件成立时执行；**`else`** 在不成立时执行；**`elif`** 处理中间的多种情况。一串判断里只会走其中一个分支。
- **缩进**决定哪些代码属于 `if`；`if`/`elif`/`else` 后面都要跟英文冒号 `:`。
- 用 **`and`（并且）、`or`（或者）、`not`（取反）** 组合多个条件。

现在程序会做一次选择了。但如果要做同一件事很多次呢？比如打印 100 遍"我错了"，难道要写 100 个 `print`？下一章，我们学**循环**——让电脑不厌其烦地重复干活，一句话就能让它跑上千万遍。翻页吧！
