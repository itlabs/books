附录 C 继续往前走

你翻到了这本书的最后一页。

停下来，回头看看走了多远——还记得第 1 章那个战战兢兢敲下 print("你好，世界！") 的你吗？那时候连英文括号和中文括号的区别都要小心翼翼。而现在，你已经能让电脑记住信息、做判断、重复干活、和你对话，甚至用上了 AI。你不再只是电脑的使用者，你成了能指挥它的人。 这是很多大人都不会的本领，而你在十几岁就学会了。

这一节不教新东西，只想和你聊聊：接下来，可以去哪儿。

先回头看看，你已经会了这些

别小看自己走过的路。这本书里，你其实掌握了编程最核心的那几块积木：

  • 让电脑输出和输入：print 说话，input 听你说话。
  • 变量：给信息起名字、装进"盒子"记住它。
  • 数据类型：文字（字符串）、整数、小数、真假值……知道它们不能乱混。
  • 条件判断：用 if / else 让程序会"看情况办事"。
  • 循环：用 for 和 while 让电脑不厌其烦地重复劳动。
  • 列表和字典：把一大堆数据整整齐齐地收纳起来。
  • 函数：把一段本领打包，起个名字，随时调用。
  • 和 AI 对话：用代码调用大模型，做出会"思考"的程序。

这些不是零散的知识点，而是几乎所有编程都在用的通用积木。换句话说：你已经有了地基。往后无论盖什么样的房子，用的都是这套地基。

小贴士：如果上面哪一条你觉得还有点虚，别担心，也别觉得非得"全懂了"才能往下走。真正的熟练不是靠读书读出来的，是靠动手写、写到手熟练出来的。带着现在的水平往前闯，反而会在实战中把它们越用越透。

往哪儿走？几条路给你选

编程像一片大大的游乐园，你现在拿到了通票。下面几个方向，每条都很好玩，挑你最心动的先去。

1. 把 Python 学得更深

这本书是入门，Python 本身还有很多好东西没讲：怎么把代码整理成一个个"模块"、怎么读写文件把数据永久保存、怎么用现成的"库"（别人写好的工具包）几行代码办大事。继续深挖 Python，是最顺的一条路，因为你已经熟悉它的脾气了。

2. 做更大的游戏

如果你喜欢书里的小游戏，可以学 Pygame——一个专门用来做游戏的 Python 库。用它你能做出有画面、有声音、能用键盘鼠标操作的真正的小游戏：躲避障碍、打飞机、走迷宫……把你脑子里的游戏点子做出来，成就感爆棚。

3. 更深入地了解 AI

你已经尝过用 AI 写程序的甜头了。往下可以了解：AI 到底是怎么"学会"东西的、怎么让 AI 帮你处理更复杂的任务、怎么把 AI 装进你自己的应用里。这是当下最火、也最有想象力的方向之一。

4. 数据分析：让数字讲故事

现实世界里到处是数据——你们班的考试成绩、你最爱的球队的战绩、一年的天气变化。数据分析就是用代码从一堆数字里找出规律，还能画成漂亮的图表。Python 在这个领域是绝对的主角（关键词：pandas、matplotlib）。喜欢数学和探究的同学会特别上瘾。

5. 做网站

想让全世界都能打开你做的东西？那就学网站开发。前端（网页长什么样）会用到 HTML、CSS、JavaScript；后端（网页背后的逻辑）Python 也能干（关键词：Flask、Django）。做一个自己的小网站发给朋友，是件很酷的事。

小贴士：别想着几条路一起走，那样容易样样都浅。 挑一个你现在最兴奋的，扎进去做几个东西，做出感觉了再说。方向之间是相通的——学深了任何一条，换到别的方向都会快很多。

最好的学习方法：做项目

我要告诉你一个很多人很晚才明白的秘密：

小贴士：光看书、光看视频，学不会编程；只有亲手做东西，才能真学会。 看别人骑车一百遍，不如自己上车摔两跤。代码也一样——你敲过、错过、改对过的，才真正长在你脑子里。

所以从今天起，给自己一个习惯：每学一点新东西，就用它做一个小玩意儿。 哪怕很简陋、很傻，也比只在脑子里"我懂了"强一百倍。

关于练习和做项目，给你几条实在的建议：

  • 从小到不能再小开始。别一上来就想做"我的世界"。先做一个能跑的小东西，再一点点加功能。能跑的小程序 > 完美的空想。
  • 允许自己抄和查。卡住了去搜、去问 AI、去看别人的代码，这不叫作弊，这就是程序员的日常工作方式。
  • 做完了给别人看。发给家长、同学、朋友。被人用、被人夸（或吐槽）是坚持下去的最大动力。
  • 找练习平台刷手感。网上有很多适合初学者的编程练习网站，把题目当小游戏闯关，能快速熟练语法。让家长帮你一起找找中文的、适合青少年的平台。
  • 建一个自己的"代码百宝箱"文件夹。把每个做过的小程序都留着。过几个月回头看，你会惊讶自己进步了这么多。

给你的 10 个项目点子

不知道做什么？这里有 10 个，都在你现在的能力范围之内（用书里学过的东西就能做）。从上往下大致由易到难，挑顺眼的开工：

  1. 每日格言机：程序里存一堆励志/搞笑的句子，每次运行随机送你一句。（随机 + 列表）
  2. 单位换算器：输入身高体重，帮你算 BMI；或人民币换算成别的单位。（输入 + 运算）
  3. 猜数字升级版：在书里的基础上加"最多猜几次""记录最好成绩""难度选择"。（循环 + 条件）
  4. 待办清单：能添加、查看、划掉今天要做的事。（列表 + 循环）
  5. 成绩管理小助手：录入各科成绩，算总分、平均分，判断优良中差。（字典 + 判断）
  6. 石头剪刀布：和电脑对战，记录几胜几负。（随机 + 条件 + 循环）
  7. 密码生成器：按你要的长度，随机生成一个又强又难猜的密码。（随机 + 字符串）
  8. 文字冒险游戏：像"你走到岔路口，往左还是往右？"这样，靠选择推进剧情。（条件 + 循环）
  9. 背单词小卡片：把要背的单词和意思存进去，随机抽考你。（字典 + 随机）
  10. AI 聊天小伙伴：用后面学的 AI 本领，做一个能陪你聊天、回答问题的小助手。（AI 调用）

小贴士：这份清单最好的用法，是照着做完一个，就自己给它加一个书上没有的新功能。"给自己出题"这件事本身，就是从"学生"变成"创作者"的分水岭。等哪天你的点子比这份清单还多，你就真的入门了。

最后，几句心里话

写代码这条路上，你一定会遇到怎么都跑不通的程序、看了半天也不懂的报错、卡住好几天的难题。这时候我想让你记住：

每一个厉害的程序员，都曾经是那个对着报错发愁的新手。 没有人天生就会。他们只是比别人多试了一次、多改了一遍、在想放弃的时候又坐回了电脑前。你会的东西，最初对他们来说也一样陌生。

你已经证明了自己能做到——你读完了这整本书，写出了会说话、会判断、会思考的程序。这份耐心和好奇心，比任何一门具体的语法都珍贵，它会带你走到很远的地方。

编程最迷人的地方，从来不是"学会了 Python"，而是那个念头："我脑子里想的东西，我能亲手把它做出来。" 从今天起，这个世界上又多了一个能创造的人。

去做点什么吧。做个游戏，做个工具，做个连你自己都觉得好玩的小东西。世界很大，代码是你的画笔。

我们，代码的世界里再见。🚀
