第 2 章 会记东西的盒子——变量与数据类型

本章你会做出什么

上一章，你让电脑说了话。但那时候电脑说完就忘，什么也记不住。这一章，我们要给程序装上"记忆"：用变量把名字、年龄、身高这些信息存起来，随时拿出来用。章末你会做一张"个人信息卡"程序，把自己的资料存进变量，再漂漂亮亮地打印出来。

成品预览：

========== 个人信息卡 ==========
姓名：小明
年龄：13 岁
身高：1.58 米
明年你就 14 岁啦！
================================

电脑的"记性"：贴标签的盒子

想象你有一排空盒子。你往一个盒子里放进数字 13，然后在盒子外面贴一张标签，写上 age（英语"年龄"）。以后你只要说"把 age 盒子里的东西拿出来"，就能得到 13。

这个"贴了标签的盒子"，就是编程里的变量（variable）。变量是用来存放数据的容器，每个变量有一个名字（就是标签）和一个值（就是盒子里装的东西）。

为什么叫"变"量？因为盒子里的东西是可以换的——今天装 13，明天你过生日了，可以把它换成 14。名字不变，里面的值能变，所以叫"变量"。

图 2-1　变量就像贴了标签的盒子：标签是名字，盒子里装着值，值可以换

赋值：把东西放进盒子

怎么把东西放进盒子？用一个等号 =。看这行：

Python 
# variable.py
age = 13
print(age)

运行结果：

13

这里的 = 叫赋值（assignment），意思是"把右边的东西装进左边的盒子"。所以 age = 13 读作："把 13 装进名叫 age 的盒子里。"

小贴士：编程里的 = 和数学里的"等于"不是一回事！数学里 age = 13 是在说"两边相等"；编程里它是一个动作——"把右边放进左边"。方向是从右往左的，别读反了。

装好之后，下一行 print(age) 里的 age 没有引号，电脑就知道："这不是要原样打印的文字，而是一个盒子的名字，我得把盒子里的东西拿出来。"于是它打印出 13。

对比一下有没有引号的区别：

Python 
print("age")   # 有引号，原样打印这三个字母
print(age)     # 没引号，打印盒子里的值

运行结果：

age
13

看出来了吧：有引号的 "age" 是一串普通文字，没引号的 age 是在喊盒子的名字。

盒子里的东西可以换

变量最厉害的地方，是里面的值随时能换。再装一次，旧的就被顶掉了：

Python 
# variable.py
score = 60
print(score)   # 现在盒子里是 60

score = 95     # 换成 95，旧的 60 没了
print(score)   # 现在盒子里是 95

运行结果：

60
95

盒子还是那个 score 盒子，但里面的东西从 60 变成了 95。这就是"变量会变"的意思。

给盒子起名字：命名规则

标签可以随便写吗？不行，有几条规矩必须守：

  • 只能用英文字母、数字、下划线 _，不能有空格和其他符号。
  • 不能用数字开头。age1 可以，1age 不行。
  • 区分大小写。age、Age、AGE 是三个不同的盒子。
  • 不能用 Python 已经占用的词（比如 print、if 这些，以后会认识）。

除了这些硬规矩，还有两条"好习惯"，会让你的代码好懂很多：

  • 起有意义的名字。存年龄就叫 age，别叫 x 或 a。过几天你自己回头看，x 是啥你都忘了。
  • 多个单词用下划线连起来，全小写。比如玩家名字叫 player_name，最高分叫 high_score。这在 Python 里叫"蛇形命名"，因为下划线像蛇趴在字母中间。
Python 
player_name = "小红"    # 好名字：一看就懂
high_score = 9999       # 好名字
x = 9999                # 能跑，但过几天你就忘了 x 是啥

小贴士：变量名一律用英文，但注释用中文——这是本书的约定。别用拼音（像 nianling），也别用中文当变量名，虽然 Python 3 勉强支持中文变量名，但很容易出各种问题，不推荐。

三种基本数据类型

盒子里能装不同种类的东西。这一章先认识最常用的三种，编程里叫数据类型（data type）：

1. 字符串（string）——一串文字。用引号包起来，单引号双引号都行。

Python 
name = "小明"
city = '北京'

2. 整数（integer，简称 int）——没有小数点的数，比如年龄、分数、个数。不加引号。

Python 
age = 13
score = 95

3. 浮点数（float）——带小数点的数，比如身高、体重、价格。"浮点"是它的学名，你就理解成"小数"。同样不加引号。

Python 
height = 1.58
price = 9.9

这里要特别小心："13" 和 13 长得像，其实完全不同。前者是字符串（一串文字，恰好长得像数字），后者是整数（真正的数字，能拿来算术）。就像照片上的苹果不能吃，只有真苹果能吃。这个坑第 3 章还会专门讲。

用 type() 查看类型

分不清一个盒子里装的是哪种类型？Python 给了你一个"照妖镜"工具：type()（英语"类型"），把变量放进它的括号里，它就告诉你类型是什么。

Python 
# type_check.py
name = "小明"
age = 13
height = 1.58

print(type(name))
print(type(age))
print(type(height))

运行结果：

<class 'str'>
<class 'int'>
<class 'float'>

str 是 string（字符串）的缩写，int 是整数，float 是浮点数。class（类别）你现在不用管，只看后面那个单词就行。以后你拿不准某个东西是什么类型，就用 type() 照一照。

让数字算起来

数字类型最好玩的地方，就是能做运算。Python 就是个超级计算器，认识这几个符号：

Python 
# math.py
print(7 + 2)    # 加法：9
print(7 - 2)    # 减法：5
print(7 * 2)    # 乘法：14（乘号是星号 *，不是 x）
print(7 / 2)    # 除法：3.5
print(7 // 2)   # 整除：3（只要商，不要小数）
print(7 % 2)    # 取余：1（除完剩下的余数）
print(7 ** 2)   # 乘方：49（7 的 2 次方，就是 7×7）

运行结果：

9
5
14
3.5
3
1
49

有几个可能对你来说是新面孔：

  • * 乘法：键盘上的星号，不是字母 x。
  • / 除法：结果永远是浮点数，8 / 2 得到的是 4.0 而不是 4。
  • // 整除：只要"商"，把小数部分砍掉。7 // 2 得 3（7 里面装得下 3 个 2）。
  • % 取余：这个叫"取余数"，7 % 2 得 1（7 除以 2，商 3 余 1，它要的就是那个余 1）。取余非常有用，比如判断一个数是不是偶数——偶数除以 2 余数一定是 0。
  • ** 乘方：两个星号，表示"几次方"。2 ** 10 就是 2 的 10 次方。

变量当然也能参与运算，就像它本来就是个数字：

Python 
# math.py
price = 8      # 一个苹果 8 元
count = 3      # 买 3 个
total = price * count   # 总价 = 单价 × 数量
print(total)

运行结果：

24

Python 先把 price 和 count 盒子里的值拿出来（8 和 3），算出 24，再装进新盒子 total。

字符串也能"相加"

那字符串能不能用 +？能，但它不是"算数"，而是把两串文字接在一起，这叫拼接（concatenation）。

Python 
# string_join.py
first = "小"
last = "明"
full_name = first + last
print(full_name)

运行结果：

小明

+ 把 "小" 和 "明" 粘成了 "小明"。想中间加个空格或别的字，就再拼一段：

Python 
greeting = "你好，" + "小明" + "！"
print(greeting)

运行结果：

你好，小明！

小贴士：+ 会看两边是什么类型。两个数字相加是算术（3 + 5 得 8），两个字符串相加是拼接（"3" + "5" 得 "35"）。但一个数字加一个字符串会报错——后面"常见错误"里会讲。

f-string：拼字符串的神器

用一堆 + 拼字符串又长又容易错。Python 有个特别好用的写法，叫 f-string（f 是 format"格式化"的意思）。做法是：在引号前面加一个字母 f，然后把变量用大括号 {} 直接嵌在文字里。

Python 
# fstring.py
name = "小明"
age = 13
print(f"你好，我叫{name}，今年{age}岁。")

运行结果：

你好，我叫小明，今年13岁。

看多方便！大括号 {name} 会被自动替换成盒子里的值。你不用管引号、不用一堆加号，想插哪个变量就 {它的名字}。

大括号里甚至能直接做运算：

Python 
age = 13
print(f"明年你就 {age + 1} 岁了。")

运行结果：

明年你就 14 岁了。

从今往后，本书拼字符串基本都用 f-string，它是你最常用的工具之一，一定要记牢：引号前加 f，变量放进 {}。

类型转换：int() / str() / float()

前面说过 "13"（字符串）和 13（整数）不一样。有时候我们手上是一种，却需要另一种——比如想把两个数字字符串加起来当算术。这时要"变个身"，用这三个工具：

  • int(x)：把 x 变成整数。
  • float(x)：把 x 变成浮点数（小数）。
  • str(x)：把 x 变成字符串。
Python 
# convert.py
a = "10"          # 这是字符串，不是数字
b = "5"

print(a + b)              # 字符串拼接，得到 "105"
print(int(a) + int(b))    # 先转成整数再相加，得到 15

运行结果：

105
15

看出区别了吗？a + b 是把两串文字 "10" 和 "5" 粘成 "105"；而 int(a) + int(b) 先把它们变成真正的数字 10 和 5，再相加得到 15。同样一对数据，转不转类型，结果天差地别。

反过来，数字要和字符串拼接，得先把数字 str() 成字符串（不过用 f-string 就免了这一步，f-string 会自动帮你转）：

Python 
score = 95
# print("我考了" + score + "分")   # 这样会报错！数字不能直接和字符串相加
print("我考了" + str(score) + "分")  # 先转成字符串就好了
print(f"我考了{score}分")             # 或者干脆用 f-string，最省事

运行结果：

我考了95分
我考了95分

小贴士：int("abc") 这种转换会报错，因为 "abc" 根本不是数字，没法变成整数。int() 只能转那些"长得像整数"的字符串。

完整代码：个人信息卡

把这一章学的东西全用上，做一张个人信息卡。新建文件 info_card.py：

Python 
# info_card.py

# 第一步：把个人信息存进变量（贴标签的盒子）
name = "小明"          # 字符串：名字
age = 13               # 整数：年龄
height = 1.58          # 浮点数：身高（米）

# 第二步：用 f-string 把信息拼成漂亮的卡片打印出来
print("========== 个人信息卡 ==========")
print(f"姓名：{name}")
print(f"年龄：{age} 岁")
print(f"身高：{height} 米")
print(f"明年你就 {age + 1} 岁啦！")   # 大括号里直接算出明年年龄
print("================================")

运行结果：

========== 个人信息卡 ==========
姓名：小明
年龄：13 岁
身高：1.58 米
明年你就 14 岁啦！
================================

试着把 name、age、height 三个盒子里的值换成你自己的，再运行一次。你会发现：只要改最上面那几个变量，下面的打印会全部跟着变——这就是变量的威力。信息只在一个地方存，用的地方全都自动同步。

常见错误

1. 变量还没赋值就使用

Python 
print(nickname)

报错：

NameError: name 'nickname' is not defined

NameError（名字错误）表示电脑不认识 nickname 这个盒子——因为你还没往这个名字里装过东西。必须先赋值，再使用。 就像你不能从一个还没造出来的盒子里拿东西。

2. 变量名和字符串搞混（该去引号的加了引号）

Python 
name = "小明"
print("name")     # 打印的是字母 name，不是小明

运行结果：

name

没报错，但结果不对。你想打印盒子里的值，就别加引号：print(name)；或者用 f-string：print(f"{name}")。

3. 数字直接和字符串相加

Python 
age = 13
print("我今年" + age + "岁")

报错：

TypeError: can only concatenate str (not "int") to str

TypeError（类型错误）在说：字符串只能和字符串拼接，不能直接粘一个整数。解决办法有两个：用 str(age) 把数字转成字符串，或者干脆用 f-string：print(f"我今年{age}岁")。

4. 用中文的等号或标点

Python
age ＝ 13

报错（SyntaxError 语法错误）。这里的 ＝ 是中文全角等号。写代码记得切英文输入法，= 要用英文半角的。

动手挑战

★ 挑战一：我的宠物档案 用变量存下一只宠物（真的或想象的）的名字、年龄、体重（用小数），然后用 f-string 打印成一段介绍，比如"我的猫叫咪咪，今年 2 岁，体重 4.5 千克。"

★★ 挑战二：小小计算器 用两个变量存两个数字，然后打印出它们的和、差、积、商四个结果。要求每行输出都用 f-string，长这样："10 加 3 等于 13"。

★★★ 挑战三：秒变小时 用一个变量存一个秒数（比如 seconds = 5000），算出它等于几分几秒，打印成"5000 秒等于 83 分 20 秒"。（提示：几分钟用整除 //，剩下几秒用取余 %。）

本章小结

这一章你给程序装上了记忆，回顾一下：

  • 变量是"贴了标签的盒子"，用来存放数据；名字是标签，值是里面装的东西。
  • 用 = 赋值，把右边的值装进左边的盒子；它是个动作，不是数学里的"等于"。
  • 变量名用英文字母、数字、下划线，不能数字开头，区分大小写，起有意义的名字。
  • 三种基本类型：字符串 str（带引号的文字）、整数 int、浮点数 float（小数）。用 type() 查看类型。
  • 数字运算：+ - * /，还有整除 //、取余 %、乘方 **。
  • 字符串用 + 拼接；更方便的是 f-string：f"你好{name}"，变量放进 {}。
  • 类型转换：int()、float()、str()，在数字和文字之间变身。
