# 附录 A 安装 Python 和写代码的工具

这本书是网页版，几乎每段代码右上角都有绿色的 **▶ 运行** 按钮，你不用装任何东西也能一路学下去。但如果你想成为真正的程序员，就该在自己的电脑上装好 Python——就像学画画的人总要有属于自己的画笔和颜料。

这一节我会手把手带你装好两样东西：

1. **Python 本身**（能运行代码的"引擎"）
2. **VS Code 编辑器**（写代码的"文具"）

> 小贴士：这一节最适合和爸爸妈妈一起看。装软件时可能需要输入电脑的开机密码，或者点几下"允许"，有大人在旁边会顺利很多。整个过程大概 15 分钟，别着急，一步一步来。

全书大部分内容 Windows 和 Mac（苹果电脑）都一样，遇到不一样的地方，我会分开写清楚。

---

## 第一步：下载并安装 Python

### 1. 打开官网

用浏览器打开 Python 的官方网站：

```
https://www.python.org
```

> 安全提醒：一定要从官方网站 **python.org** 下载，别在乱七八糟的网站上下"Python 安装包"，那些很可能带病毒。认准网址里的 `python.org`。

把鼠标移到导航栏的 **Downloads（下载）** 上，网站通常会自动认出你的系统，给你一个大大的黄色按钮，比如"Download Python 3.13.x"。点它就开始下载。（`3.13.x` 里的 `x` 是小版本号，你看到的数字可能比这大，没关系，只要是 3 开头的最新版就行。）

### 2. Windows 用户：安装时务必勾选这一项！

下载完成后，双击那个 `.exe` 安装文件。安装窗口一打开，**先别急着点"Install Now"！**

请先看窗口**最下面**，有一个小方框：

```
☑ Add python.exe to PATH
```

（意思是"把 Python 加到 PATH 里"。）

**一定要把这个方框打上勾！** 然后再点上面的 **Install Now（现在安装）**。

> 小贴士：**这一步是全书最容易出错、也最重要的一步。** PATH 你可以理解成电脑的"通讯录"——勾了它，以后你在任何地方喊一声"python"，电脑都知道去哪儿找它。如果不勾，电脑就会一脸茫然地说"我不认识 python 这个命令"。很多新手卡在这里，所以我啰嗦提醒你：**先勾 Add to PATH，再点 Install。**

安装完成后，点 **Close（关闭）** 就好。

### 3. Mac 用户：一路点"继续"就行

Mac 上简单得多：下载下来的是一个 `.pkg` 文件，双击它，然后一路点 **继续（Continue）→ 同意 → 安装（Install）**，中途可能要你输一次开机密码。装完关掉窗口即可。Mac 不需要你手动勾选什么，它会自动配置好。

## 第二步：验证 Python 装好了没

装完怎么知道成功了？我们让电脑自己报一下"我几岁了"（也就是版本号）。这要用到**终端**——一个可以直接给电脑打字下命令的黑框框。

### 怎么打开这个"黑框框"

**Windows：打开"命令提示符"**

点开始菜单（屏幕左下角的窗户图标），直接用键盘打字 `cmd`，屏幕上会出现"命令提示符"，点它打开。你会看到一个黑色（或蓝色）的窗口，里面有一行字，光标在闪。

**Mac：打开"终端"**

按 `Command（⌘）+ 空格` 打开搜索框，输入 `终端` 或 `Terminal`，回车打开。

> 小贴士：**终端（Terminal）** 和 Windows 的**命令提示符（Command Prompt）** 是一回事，都是"你直接跟电脑打字对话"的地方。本书后面说"打开终端"，Windows 同学就理解成"打开命令提示符"。它看起来很酷很黑客，其实一点都不难。

### 输入验证命令

在那个框框里，敲下面这行，然后按回车：

```bash
python --version
```

如果一切正常，它会回你一行，比如：

```
Python 3.13.1
```

看到 `Python` 加一串数字，就说明**你成功了！** Python 已经住进你的电脑了。

> 小贴士：Mac 用户如果上面这行没反应，试试把 `python` 换成 `python3`：
>
> ```bash
> python3 --version
> ```
>
> Mac 系统里 `python3` 才是我们刚装的那个。记住这个小区别就好。

如果它没有报出版本号，而是说了句"不是内部或外部命令"之类的话——别慌，跳到本附录最后的"常见安装问题"看看。

## 第三步：安装 VS Code 编辑器

Python 是"引擎"，但你还需要一个舒服的地方**写**代码。我们用 **VS Code**（全名 Visual Studio Code），它免费、好用，全世界的程序员都在用，特别适合新手。

> 小贴士：写代码为什么不用记事本或 Word？因为专业编辑器会**给代码上色**（不同的东西显示成不同颜色）、**帮你对齐**、**发现拼写错误**，就像一个随时在旁边的小助手。用它写代码，会舒服一百倍。

### 1. 下载安装

打开官网：

```
https://code.visualstudio.com
```

点大大的下载按钮，选对应你系统的版本（网站一般会自动认出来）。

- **Windows**：下载 `.exe`，双击，一路"下一步"。中间有个"选择附加任务"的界面，建议把"添加到 PATH"和几个"添加右键菜单"的勾都打上，方便以后使用。
- **Mac**：下载的是压缩包，解压后会得到一个 **Visual Studio Code** 图标，把它**拖进"应用程序（Applications）"文件夹**，就算装好了。

### 2. 装中文界面

VS Code 默认是英文的，我们把它变成中文，看着更亲切。

1. 打开 VS Code。
2. 看左边一竖排图标，找到那个像**四个小方块**的图标（叫"扩展 Extensions"），点它。
3. 上方的搜索框里输入：`Chinese`
4. 找到 **"Chinese (Simplified) 简体中文"** 这个扩展，点它旁边的蓝色 **Install（安装）** 按钮。
5. 装完右下角会弹出一个提示，问你要不要重启，点 **Restart（重启）**。重启后界面就变中文啦。

### 3. 装 Python 扩展

同样在"扩展"面板里：

1. 搜索框输入：`Python`
2. 找到微软官方出的那个 **Python** 扩展（作者是 Microsoft，下载量最高、排最前面的那个），点 **安装**。

这个扩展会让 VS Code"看懂"Python 代码：帮你上色、提示错误、还会给你一个方便的运行按钮。

> 小贴士：扩展就像给手机装 App——给 VS Code 装上"Python 扩展"，它就多了懂 Python 的本领。装好这两个扩展就够开始了，别的以后需要再说。

## 第四步：新建文件、写代码、跑起来

万事俱备，我们来完整走一遍：从新建文件到看见结果。

### 1. 新建一个文件夹放你的代码

先在电脑上建一个专门放代码的文件夹，比如在"文档"里建一个叫 `python_code` 的文件夹。以后你写的程序都放这儿，整整齐齐。

在 VS Code 里点 **文件（File）→ 打开文件夹（Open Folder）**，选中你刚建的这个文件夹打开。

### 2. 新建 .py 文件

在 VS Code 左边的文件列表区，把鼠标移上去会出现几个小图标，点那个**"新建文件"**的图标（一张带加号的纸），给文件起个名字：

```
hello.py
```

> 小贴士：**文件名一定要以 `.py` 结尾**，这是在告诉电脑"这是一份 Python 代码"，就像图片以 `.jpg` 结尾一样。文件名用英文，别用空格，比如 `my_game.py` 就很好。

### 3. 写几行代码

在文件里敲下这一行（照第 1 章学的）：

```python-norun
# hello.py
print("你好，世界！")
print("我的电脑装好 Python 啦！")
```

按 `Ctrl + S`（Mac 是 `⌘ + S`）保存。

### 4. 运行它——两种办法

**办法一：点运行按钮（最简单）**

装了 Python 扩展后，VS Code 编辑器**右上角**会出现一个**三角形的运行按钮 ▶**。点它，代码就跑起来了。下方会弹出一个叫"终端"的小窗口，显示运行结果：

```
你好，世界！
我的电脑装好 Python 啦！
```

**办法二：在终端里用命令运行（更接近真实开发）**

在 VS Code 里点 **终端（Terminal）→ 新建终端（New Terminal）**，打开底部的终端窗口，输入：

```bash
python hello.py
```

（Mac 若不行就用 `python3 hello.py`。）回车，同样会看到结果。

> 小贴士：`python hello.py` 这句话的意思是"喊 Python 来，运行 hello.py 这个文件"。运行前要确认终端当前就在文件所在的文件夹里——在 VS Code 里"打开文件夹"后新建的终端会自动待在正确的位置，所以推荐你就用 VS Code 里的终端。

两种办法结果一样，你喜欢哪种用哪种。**恭喜你，现在你有了一套完整的编程环境！** 这本书剩下的所有代码，你都可以在这里亲手敲、亲手跑了。

## 常见安装问题

装软件遇到问题太正常了，几乎每个程序员都经历过。这里列几个最常见的，对号入座就行。

### 1.「'python' 不是内部或外部命令」（或 Mac 上「command not found: python」）

这是**最常见**的问题，几乎百分之百是因为——**装 Python 时忘了勾"Add Python to PATH"**。

电脑不知道去哪儿找 python，所以喊它没反应。解决办法：

- **最省事的办法**：重新运行一遍 Python 安装程序（就是那个 `.exe`），这次**记得勾上最下面的"Add python.exe to PATH"**，选择"Modify（修改）"或干脆重装一遍。装完**关掉终端再重新打开**（很重要，不然它记不住新设置），再试 `python --version`。
- **Mac 用户**：先试试 `python3 --version`，Mac 上多数时候用 `python3` 就对了。

> 小贴士：改完 PATH 一定要**把终端/命令提示符窗口整个关掉，再重新开一个**。终端只在打开的那一刻读一次"通讯录"，你中途改了它不知道，得重开才生效。这个小细节能省你半小时的抓狂。

### 2. 点了运行按钮，却弹出一堆看不懂的东西

先深呼吸。看看弹出的终端窗口**最后一行**，如果里面有 `Error`（错误）字样，那不是装的问题，而是代码本身有小毛病——翻到[附录 B](appendix-b.md)，按报错类型查怎么修。报错不可怕，它是电脑在帮你找问题。

### 3. VS Code 找不到运行按钮 ▶

十有八九是 **Python 扩展没装好**。回到"扩展"面板（四个小方块图标），搜 `Python`，确认微软官方那个已经显示"已安装"。装好后可能要**重启一下 VS Code**（整个关掉再打开）才会出现运行按钮。

### 4. 下载特别慢

Python 和 VS Code 的官网服务器在国外，有时候下载会慢。可以：多等一会儿；或者换个网络（比如手机热点）试试；实在不行，让家长帮忙找找国内的镜像站，但**认准是官方版本**，别下到来路不明的安装包。
