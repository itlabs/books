## 电脑其实很"笨"

在开始之前，先记住一件事：**电脑非常听话，但也非常笨。** 它只会做你明确告诉它的事，一字不差。你说"打印你好"，它就打印"你好"；你少写一个符号，它可能就完全看不懂。

所以**编程（programming）**，说白了就是：用一种电脑能听懂的语言，把你想让它做的事，一步一步、清清楚楚地写下来。我们用的这门语言叫 **Python（读作"派森")**，它是世界上最受欢迎的编程语言之一，长得很像英语，特别适合入门。

我们写下的这些"命令"，合起来就叫**代码（code）**；一整份能完成某件事的代码，叫一个**程序（program）**。

## 第一行代码：让电脑说话

打开你的代码编辑器（比如 VS Code，见附录 A），新建一个文件，命名为 `hello.py`。

> 小贴士：Python 程序的文件名都以 `.py` 结尾，就像图片以 `.jpg` 结尾一样。这是在告诉电脑"这是一份 Python 代码"。

在文件里敲下这一行：

```python
# hello.py
print("你好，世界！")
```

保存，然后运行它（怎么运行见附录 A，通常是在终端里输入 `python hello.py`）。

运行结果：

```
你好，世界！
```

成功了！你刚刚写出了人生第一个程序。几乎每个程序员的第一个程序，都是让电脑说这句"你好，世界"。恭喜你，正式加入这个大家庭。

## 拆开看看这行代码

这短短一行里，藏着几个重要的东西，我们一个个认识：

```python
print("你好，世界！")
```

- **`print`**：这是一个**函数（function）**，你可以把它理解成一个"小工具"或"按钮"。`print` 这个工具的作用就是"把括号里的东西打印到屏幕上"。（print 是英语"打印"的意思。）
- **`( )` 圆括号**：你要让工具处理的东西，得放进它的括号里。就像你要用微波炉热饭，得把饭放进去、关上门。
- **`"你好，世界！"` 引号里的文字**：这段被双引号包起来的文字，在编程里叫**字符串（string）**——就是"一串字符"的意思。引号是在告诉电脑："从这里到那里，都是一段普通文字，原样打印出来就好。"

连起来读就是：**"嘿 print 工具，把'你好，世界！'这段文字打印到屏幕上。"**

> 小贴士：括号、引号这些符号，必须用**英文半角**的，不能用中文的（）「」。这是新手最常犯的错误之一，本章末尾会专门讲。

## 那个 `#` 是什么？

你注意到第一行 `# hello.py` 了吗？以 `#` 开头的行叫**注释（comment）**。

注释是**写给人看的，电脑会完全忽略它**。它就像你在作业本边上写的小笔记，用来提醒自己"这段代码是干嘛的"。本书里，每份完整代码的开头，我都会用注释写上文件名，方便你对照。

试试看，把这段加到你的文件里再运行：

```python
# 这一行是注释，电脑不会理会
print("电脑看不见上面那行字")  # 写在代码后面的注释也可以
```

运行结果：

```
电脑看不见上面那行字
```

看，注释一个字都没被打印出来。以后你写复杂代码时，注释会帮大忙。

## 让电脑多说几句

一个 `print` 只能打印一行。想让电脑说好几句话，就写好几个 `print`，它们会**从上到下依次执行**。这是编程里最重要的规律之一：**代码是一行一行、按顺序运行的。**

```python
# hello.py
print("你好，世界！")
print("欢迎来到编程的世界，新程序员！")
```

运行结果：

```
你好，世界！
欢迎来到编程的世界，新程序员！
```

电脑先执行第一行，打印第一句；再执行第二行，打印第二句。顺序不能乱——如果你把两行调换，打印出来的顺序也会跟着换。

## 动手改一改

现在轮到你了。把上面的程序改成打印你自己的话，比如：

```python
# hello.py
print("我叫小明。")
print("我今年 13 岁。")
print("这是我写的第一个程序！")
```

把名字和年龄换成你自己的，运行看看。你会发现：**代码是你说了算的**，你写什么，它就说什么。这种"我能指挥电脑"的感觉，就是编程最上瘾的地方。

## 常见错误

新手第一天最容易踩这几个坑，先认识它们，遇到就不慌：

**1. 用了中文的括号或引号**

```python-norun
print（"你好"）
```

运行会报错，最后一行大概长这样：

```
SyntaxError: invalid character '（' (U+FF08)
```

`SyntaxError` 是"语法错误"，意思是电脑看不懂你写的东西。这里的问题是括号 `（）` 是中文全角的。改成英文的 `()` 就好了。**记住：写代码时把输入法切到英文。**

**2. 引号忘了配对**

```python-norun
print("你好)
```

报错：

```
SyntaxError: unterminated string literal
```

意思是"字符串没结束"——你开了引号却没关上。引号必须一头一尾成对出现：`"你好"`。

**3. `print` 拼错了**

```python-norun
prit("你好")
```

报错：

```
NameError: name 'prit' is not defined
```

`NameError`（名字错误）表示电脑不认识 `prit` 这个词。检查一下拼写，是 `p-r-i-n-t`，五个字母，一个都不能少或错。

> 小贴士：报错不可怕，它其实是电脑在**帮你**指出问题在哪儿。多数报错的最后一行会告诉你错误类型和大概原因。看不懂的报错，去[附录 B](appendix-b.md) 查。

## 本章小结

这一章你迈出了最重要的第一步。回顾一下你学到的：

- **编程**就是用电脑能懂的语言，把要做的事一步步写清楚；电脑很听话但很笨，只做你明确说的事。
- **`print()`** 是打印文字的工具（函数），要打印的内容放在圆括号里。
- 用**双引号 `" "`** 包起来的文字叫**字符串**。
- 以 **`#`** 开头的是**注释**，写给人看的，电脑会忽略。
- 代码**从上到下按顺序一行一行执行**。
- 符号必须用**英文半角**，这是新手最常见的报错来源。
