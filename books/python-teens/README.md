# 《跟 Python 一起玩：从零开始，用游戏和 AI 学编程》

> 一本写给 7—9 年级同学的编程书。不需要任何基础，只要你愿意动手敲代码。

## 这本书讲什么

你会先学会 Python 的基本语法，然后用它做出一个又一个能玩的**游戏**；最后，你还会学着调用 **AI（人工智能）** 的编程接口，让你写的程序变得"聪明"起来——能陪你聊天，能给游戏当裁判，甚至能自己编故事。

学编程最好的方法就是做点好玩的东西。所以这本书里，几乎每一个知识点都会变成游戏里的一个零件。

## 怎么用这本书

1. **一定要动手敲。** 光看不练，学不会编程。每段代码都自己敲一遍、运行一遍。
2. **按顺序读。** 后面的章节会用到前面的知识。
3. **做挑战题。** 每章末尾的"动手挑战"是真正让你变强的地方。
4. **不怕报错。** 程序报错是常态，本书附录 B 教你怎么看懂报错。
